Linux-Befehle für WSL & Termux

Übersichtliche Sammlung gängiger Linux-Befehle inkl. virtueller Umgebungen und Session-Erstellung

ls

Zeigt den Inhalt des aktuellen Ordners an

cd ordnername

Wechselt in den Ordner ordnername

cd ..

Geht einen Ordner nach oben

cd /

Wechselt ins Root-Verzeichnis

cd ~

Wechselt ins Home-Verzeichnis

pwd

Zeigt den aktuellen Pfad an

python3 -m venv env

Erstellt eine virtuelle Python-Umgebung

source env/bin/activate

Aktiviert die erstellte virtuelle Umgebung

tmux new -s sessionname

Erstellt eine neue tmux Session mit dem Namen sessionname

tmux attach -t sessionname

Tritt der bestehenden tmux Session sessionname bei
