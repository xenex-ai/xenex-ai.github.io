===========================================
Solana Termux Installer for Linux/Android
===========================================

This script sets up a full Solana development environment inside Termux on Android/Linux.
It automatically installs all required dependencies and tools, including Solana CLI and Rust.

-------------------
Features:
-------------------
- One-command automatic setup
- Installs:
  * Rust toolchain
  * Solana CLI
  * Optional: Node.js and other dependencies
- Fully compatible with Termux (Android)
- Lightweight and mobile-friendly

-------------------
Quick Start:
-------------------
1. Open Termux (recommended from F-Droid):
   https://f-droid.org/packages/com.termux/

2. Run this command:

   curl -sSL https://raw.githubusercontent.com/xenex-ai/xenex-ai.github.io/main/packages/solana_linux-android-termux.sh | bash

The script will handle the full setup automatically.

-------------------
Optional:
-------------------
During installation, you may be prompted to install Rust and/or Node.js.
Type 'y' (yes) or 'n' (no) as needed.
